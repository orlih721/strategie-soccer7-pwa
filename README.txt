Stratégie Soccer 7 - PWA v1

Pour tester sur ton ordinateur :
1. Installer Node.js
2. Ouvrir un terminal dans ce dossier
3. npm install
4. npm run dev

Pour tester sur iPad :
1. Ton PC et ton iPad doivent être sur le même Wi-Fi
2. Dans le terminal, npm run dev -- --host 0.0.0.0
3. Sur l'iPad, ouvrir Safari avec l'adresse affichée dans le terminal
4. Dans Safari : Partager > Ajouter à l'écran d'accueil
